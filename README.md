#  Calculadora de M²

Aplicativo Android para calcular área e orçamento de revestimentos de forma rápida e simples.

##  Funcionalidades

- Cálculo de área em m² (largura × altura)
- Cálculo do valor total com base no preço por m²
- Opções de arredondamento (2 casas decimais ou para cima)
- Compartilhamento de orçamento via apps (WhatsApp, Telegram, etc.)
- Diálogo de saída com avaliação na Play Store
- Anúncios via Google AdMob
